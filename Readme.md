## 如何自定义

- `goodAuthors`, `goodNovels` 是对看过的文的屏蔽

- `badAuthors`, `badNovels` 是对不感兴趣的文的屏幕

- `badTypes` 是对晋江分类里标签的屏蔽

- `badKeywords` 是对标题里关键词的屏蔽

- `NanZhuWuCP` 是对男主无CP文的屏蔽

我列的都是一些例子，使用的话按照自己的需求修改变量就好了
